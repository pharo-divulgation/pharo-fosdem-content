Visualizations powered by Pharo

The ability to visualize data is crucial in many areas ranging from artificial intelligence to simulation and data analysis. The Pharo community maintains several packages that help in building interactive 2D and 3D visualizations. The following videos showcase some of our most popular packages.

Roassal 3

Roassal is an interactive 2D visualization engine.

Git repository: https://github.com/ObjectProfile/Roassal3 More at: http://objectprofile.com/​ http://objectprofile.com/​

















Wooden

Wooden is a 3D game engine that shows its power together with Pharo’s live-programming capabilities.

Git repository: https://github.com/woden-engine/woden​
